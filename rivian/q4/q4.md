detect aggressive braking

Rivian's autonomy team needs to identify safety issues. You are given a massive dataset of vehicle telemetry.

Input Data: A Spark DataFrame df_telemetry with millions of rows:

vin: Vehicle Identification Number (String)

ts: Timestamp (Timestamp)

velocity: Current speed in mph (Float)

The Task: A vehicle is flagged for "Aggressive Braking" if its speed drops by more than 20 mph within a 2-second window. 

Write a PySpark function to return a DataFrame of all vins and the ts where this event occurred.
